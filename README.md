### Installation
1. Create `.env` file and include following:
```
graphqlAPI = process.env.NEXT_PUBLIC_GRAPHCMS_ENDPOINT
graphcmsToken = process.env.GRAPHCMS_TOKEN
```
2. Run `npm run dev` to run both the backend server and the client side.
